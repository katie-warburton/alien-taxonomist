Instructions

We need you help!

Our scientists have just discovered some new alien organisms on Mars. They have started to classify them into taxonomies but have become overwhelmed by the sheer number of aliens. That's where you come in! We have some alien taxonomies that our scientists have started, but still more aliens that need to be classified into them. We need you to sort these aliens into the correct taxonomic category and add new categories if an appropriate one doesn't exist. Take note, small changes in appearance can mean big changes in behaviour, so make sure you pay attention and sort them as accurately as possible.

First you will do a tutorial to get familiar with the taxonomy setup. Press continue when you are ready to start.
